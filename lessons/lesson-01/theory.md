**Duración estimada**: 5 minutos  
**Público**: Estudiantes de 12 a 17 años sin experiencia previa

---

## Contenido teórico

### ¿Qué es programar?

Programar es darle instrucciones a una computadora para que haga algo.  
Igual que una receta de cocina: paso a paso, la computadora las sigue exactamente.

### ¿Qué es Java?

Java es un lenguaje de programación creado en 1995. Se usa para:

- Aplicaciones de Android (WhatsApp, Instagram en sus inicios)
- Sistemas bancarios
- Videojuegos (Minecraft está hecho en Java)
- Sitios web y mucho más

### ¿Cómo luce un programa Java?

```java
public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("¡Hola, mundo!");
    }
}
```

Puntos clave:
- `public class HolaMundo` → define la clase principal (el archivo se llama igual)
- `main` → es el punto de entrada; Java empieza a ejecutar desde aquí
- `System.out.println(...)` → imprime texto en pantalla
