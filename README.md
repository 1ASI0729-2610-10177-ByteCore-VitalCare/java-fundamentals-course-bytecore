# Fundamentos de Java para Principiantes — ByteCore

Repositorio de código fuente del curso introductorio de Java dirigido a estudiantes de secundaria (12–17 años).  
Sin instalaciones. Solo un navegador web.

## Estructura del repositorio

```
java-fundamentals-course-bytecore/
├── course-plan.md              # Plan completo del curso (artefacto principal)
└── lessons/
    ├── lesson-01/              # ¿Qué es Java y para qué sirve?
    │   ├── theory.md           # Contenido teórico + enlace a práctica online
    │   ├── starter/            # Código inicial con TODOs para el estudiante
    │   └── completed/          # Código completo de referencia
    ├── lesson-02/              # Variables y tipos de datos
    ├── lesson-03/              # Operadores y expresiones
    ├── lesson-04/              # Estructuras de control (if-else, for, while)
    ├── lesson-05/              # Métodos y entrada/salida básica
    ├── lesson-06/              # Introducción a la POO: Clases y Objetos
    └── lesson-07/              # Encapsulación y Proyecto Final
```
